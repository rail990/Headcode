  • Signal logic — red, yellow, green based on block state
  • Speed control per district — variable voltage via PWM
  • Clear upgrade path — add a Z21 later and switch to DCC mode

Everything your layout needs. Nothing it doesn't.

01 — Safety
Crash blocking DC + DCC
Trains detect each other automatically. Block occupancy sensors feed live data to the app. DC districts are cut instantly; DCC locos receive an emergency stop command. Set it up once. Let it run.
02 — Signalling
Real signal logic DC + DCC
Signals that behave like the real thing. Red, yellow, green — updated instantly as trains move through blocks on either system. No scripting. No manual triggers.
03 — Routing
Route setting & points DCC
Set a route and the app throws the right points, checks every block is clear, and locks the path until your train is through. Just like a real signalbox — but from your phone.
04 — DC Control
PWM speed control DC
Variable voltage control per district via Raspberry Pi PWM output. Smooth speed changes, direction control, and per-district power management — no DCC system required.
